This plugin allows you to insert a picture of the theme directory.

== Description ==

Insert images in theme folder.
Manipulated as easily as you would choose from Media Library.

== Installation ==

1. Upload the entire media-insert-from-themes-dir folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Menu is added on the Media Insert.

== 日本語でのご説明 ==

このプラグインはテーマフォルダ内の画像を「メディア挿入」から挿入できるようにする為のプラグインです。
使い方は簡単です。
「メディアライブラリ」と同じようにテーマフォルダ内の画像を選択し、「挿入」ボタンを押すだけです。
